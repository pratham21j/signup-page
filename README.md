#  Sign Up Page

A simple and clean **Sign Up Page UI** built using HTML and CSS.
This project focuses on creating a modern, responsive form design.

---

##  Features

* Clean and modern UI
* Responsive design (works on different screen sizes)
* Input focus effects
* Styled button with hover effects
* Simple and beginner-friendly code

---

##  Tech Stack

* HTML5
* CSS3
